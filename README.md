<p align="left">
  <strong>ZERO</strong> is a Python-based command-line intelligent assistant designed to make mathematical calculations intuitive and accessible. Unlike standard calculators, ZERO understands both mathematical symbols and natural language commands in <b>English</b> and <b>Malay</b>.
</p>


## ✨ Key Features

* Multilingual Support
    Interact with ZERO using natural phrases in either English or Malay.
* Natural Language Processing (NLP)**
    Intelligently recognizes operations like *plus*, *kali*, *tambah*, or *bahagi*.
* Advanced Mathematics
    Supports standard arithmetic plus square roots (`sqrt` or `√`), powers (`^`), factorials (`!`), and modulus (`%`).
* Interactive CLI
    A clean, conversational terminal interface featuring custom ASCII branding for a professional feel.


## 💡 How to Use

Once **ZERO** asks *"How can I help you?"*, simply type your request.

| Context | Command Example | Expected Response |
| :--- | :--- | :--- |
| **English** | "Can you tell me what is 10 plus 5?" | "The result is 15" |
| **Direct** | `20 + 4` | "The result is 24" |
| **Malay** | "Apa itu √ 144?" | "Hasilnya adalah 12" |

> [!TIP]
> **To Exit:** Simply type `Exit` to end the session.
