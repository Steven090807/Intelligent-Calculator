# 🤖 ZERO: The Intelligent Multilingual Calculator
<h4>ZERO is a Python-based command-line intelligent assistant designed to make mathematical calculations intuitive and accessible. Unlike standard calculators, ZERO understands both mathematical symbols and natural language commands in English and Malay.</h4>

✨ Features<br>
  <pre><li><b>Multilingual Support:</b> Interact with ZERO using English or Malay commands.</li>

  <li><b>Natural Language Processing:</b> Recognizes operations like plus, kali, or bahagi.</li>

  <li><b>Advanced Math:</b> Supports standard arithmetic plus square roots (sqrt or √), powers (^), factorials (!), and modulus (%).</li>

  <li><b>Interactive CLI:</b> A clean, conversational terminal interface with custom ASCII branding.</li></pre>

💡 How to Use<br>
Once ZERO asks "How can I help you?", you can simply type your request.

   Example 1: "Can you tell me what is 10 plus 5"

   Example 2: <code>20 + 4</code>

   Example 3: "Apa ini √ 144"

To Exit: Type Exit.
